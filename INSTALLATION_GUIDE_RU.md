# Инструкция по установке и запуску VPN Key Collector

## Шаг 1: Скачивание Python

1. Перейдите на официальный сайт: https://www.python.org/downloads/windows/
2. Скачайте **Windows installer (64-bit)** последней версии (Python 3.12.x или 3.13.x)
   - Примечание: Python 3.13.x не работает на Windows 7 и старее

## Шаг 2: Установка Python

1. Запустите скачанный установщик от имени **Администратора** (правый клик → "Запуск от имени администратора")
2. В окне установителя обязательно отметьте следующие галочки:
   - ✅ **Add Python to PATH** (Добавить Python в PATH) - **ОБЯЗАТЕЛЬНО**
   - ✅ **Install for all users** (Установить для всех пользователей)
   - ✅ **Associate files with Python** (Ассоциировать файлы с Python)
   - ✅ **Create shortcuts for installed applications** (Создать ярлыки)
3. Нажмите кнопку **Install Now**
4. Дождитесь завершения установки
5. Нажмите **Close**

## Шаг 3: Перезагрузка компьютера

**ВАЖНО:** После установки Python обязательно перезагрузите компьютер для применения изменений в PATH.

## Шаг 4: Подготовка папки приложения

1. Создайте папку `C:\VPN-Key-Collector`
2. Скопируйте все файлы из папки `d:\Program\RustEdit\CustomPrefabs\VPN-Key-Collector` в `C:\VPN-Key-Collector`
   - `vpn_key_app.py` - главный файл приложения
   - `README.md` - документация
   - Другие файлы проекта

## Шаг 5: Установка зависимостей

1. Откройте **PowerShell от имени Администратора**:
   - Нажмите `Win + X`
   - Выберите **Windows PowerShell (Администратор)** или **Terminal (Администратор)**

2. Перейдите в папку приложения:
   ```powershell
   cd C:\VPN-Key-Collector
   ```

3. Установите библиотеку requests:
   ```powershell
   pip install requests
   ```

## Шаг 6: Запуск приложения

1. В той же PowerShell выполните команду:
   ```powershell
   python vpn_key_app.py
   ```

2. Откроется окно приложения **VPN Key Collector**

## Инструкция по работе с приложением

### Интерфейс приложения

**Верхняя панель:**
- **Источник ключей** - выпадающий список с 25 источниками VPN ключей
- **🔍 Проверить ключи** - кнопка для запуска проверки
- **Авто-проверка (1 час)** - галочка для автоматической проверки каждый час
- **Статус** - текущий статус работы

**Центральная область:**
- Текстовое поле с результатами проверки ключей
- Показывает рабочие ключи с пингом

**Нижняя панель:**
- **📋 Копировать лучший ключ** - копирует самый быстрый ключ
- **📋 Копировать все ключи** - копирует все рабочие ключи
- **💾 Сохранить в файл** - сохраняет ключи в txt файл
- **🚀 Открыть v2rayN** - открывает VPN клиент

### Рекомендуемые источники для мобильных сетей с глушилками

Для работы на мобильных сетях с активными глушилками используйте источники **WHITE (белые списки)**:

- **WHITE CIDR (белые списки)** - Проверенные CIDR конфиги
- **WHITE CIDR All** - Все CIDR конфиги
- **WHITE SNI (белые списки)** - SNI конфиги
- **WHITE Reality Mobile** - Reality для мобильных
- **WHITE Reality Mobile 2** - Reality для мобильных (альтернатива)

### Пошаговая работа

1. **Выберите источник ключей**
   - Для обычного VPN: BLACK источники
   - Для глушилок: WHITE источники

2. **Нажмите "🔍 Проверить ключи"**
   - Приложение загрузит ключи с GitHub
   - Проверит TCP-доступность каждого сервера
   - Измерит пинг до каждого сервера
   - Отфильтрует нерабочие ключи

3. **Дождитесь завершения проверки**
   - В текстовом поле появится прогресс
   - Рабочие ключи будут отмечены ✅
   - Нерабочие ключи будут отмечены ❌

4. **Скопируйте лучший ключ**
   - Нажмите "📋 Копировать лучший ключ"
   - Ключ скопируется в буфер обмена

5. **Подключитесь через VPN клиент**
   - Откройте v2rayN (кнопка "🚀 Открыть v2rayN")
   - Вставьте ключ в v2rayN
   - Подключитесь

### Авто-проверка

Включите галочку **"Авто-проверка (1 час)"** для автоматического обновления ключей каждый час. Приложение будет автоматически проверять ключи и обновлять список.

### Сохранение ключей

Нажмите **"💾 Сохранить в файл"** чтобы сохранить все рабочие ключи в txt файл с текущей датой и временем. Файл будет создан в папке приложения.

## Решение проблем

### Python не распознается

Если после установки Python команда `python` не работает:

1. Проверьте установку:
   ```powershell
   py --version
   ```

2. Если py работает, используйте:
   ```powershell
   py -m pip install requests
   py vpn_key_app.py
   ```

3. Если не работает, переустановите Python и убедитесь что отмечена галочка **Add Python to PATH**

### Ошибка при установке requests

Если pip не работает:
```powershell
python -m pip install --upgrade pip
python -m pip install requests
```

### Приложение не запускается

Убедитесь что:
- Python установлен правильно
- Библиотека requests установлена
- Файл vpn_key_app.py находится в папке C:\VPN-Key-Collector
- Запускаете от имени администратора (если требуется)
